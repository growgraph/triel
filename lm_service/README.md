## Definitionstoken space : $\alpha \in \mathcal{A}$trees of tokens comprise candidate space: $\kappa (\alpha) \in \mathcal{\Kappa}$ordered triples of tokens comprise triple space: $\tau \in \Tau(\Kappa)$, in such a way that there are projections: - source : $\sigma :\Tau \to \Kappa$- relation : $\rho :\Tau \to \Kappa$- target : $\pi :\Tau \to \Kappa$Define an extension of triple space by adding itself to candidate space: $\Kappa^* = \Kappa \bigcup T$.
